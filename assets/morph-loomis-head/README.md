# Morph + Loomis Head Asset

Main GLB:

- `morph_loomis_head_arkit_oculus.glb`

This is a compact procedural prototype asset for Morph Head Studio.

Contents:

- head + neck mesh
- Loomis construction guide lines
- transparent side plane guides
- 72 named morph targets
- 52 ARKit names
- 15 Oculus / OVR viseme names
- 5 helper names: `mouthOpen`, `mouthSmile`, `eyesClosed`, `eyesLookUp`, `eyesLookDown`

License / usage:

- CC0 / user-created procedural prototype geometry
- safe for internal app testing and iteration

Honesty note:

This is a lightweight repo-safe prototype, not a photoreal production face rig. It is meant to prove import, morph target naming, morph slider behavior, and Loomis overlay behavior inside the app.
